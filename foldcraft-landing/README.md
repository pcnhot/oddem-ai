# Foldcraft — Hero Landing

A fullscreen hero landing page for the creative studio **Foldcraft**, built with
React, Tailwind CSS, and Lucide React icons.

> Note: this is a standalone web demo living in a subdirectory. The repository
> root hosts the unrelated ODEEM Flutter project.

## Features

- Single viewport-height hero section with a looping, muted background video.
- Responsive navbar with desktop links and a mobile hamburger menu.
- Full-screen mobile menu overlay with animated open/close transitions.
- Staggered `fadeSlideUp` entrance animations on the hero text and CTAs.
- Geist font (Google Fonts) applied via Tailwind's `font-geist` utility.

## Stack

- React 18 + Vite
- Tailwind CSS 3
- lucide-react (`ArrowRight`, `Menu`, `X`)

## Getting started

```bash
cd foldcraft-landing
npm install
npm run dev      # start the dev server
npm run build    # production build
npm run preview  # preview the production build
```
